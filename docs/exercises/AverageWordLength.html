Workshop Exercise: Average Word Length

In this exercise, you will modify the AverageWordLength project. This project computes the average length of words based on their first character. After execution, it will report the average length of words that begin with 'a', 'b', and so forth.

The existing project is based on the standard word count example, and does not use combiners.

The tasks

  • Task 1: Modify algorithm to take advantage of combiners

    • Do it in the easiest way possible (for example, pack complex data into a Text object).

  • Task 2: Rewrite using custom Writables

  • Task 3: Experiment with various techniques discussed in The "Art of Using Combiners" section.

Getting started

If you update the exercises, the starter source for this will appear in the shakespeare project. You can update your own copy by running: 

        $ cd ~/git
        $ ./update-exercises

The starter files will appear in ~/git/exercises/shakespeare/stub-src/src/wordlen/. You may also need to refresh (F5) the project in Eclipse to see the updates.
